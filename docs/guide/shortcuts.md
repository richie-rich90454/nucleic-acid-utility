# Keyboard Shortcuts

The following keyboard shortcuts are available when the application page is focused.

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Run the selected conversion |
| `Ctrl+Shift+C` | Copy the conversion result to the clipboard |
| `Ctrl+Backspace` | Reset all fields (input, output, and visualization) |

These shortcuts operate globally within the page context. They do not require a specific element to be focused.
